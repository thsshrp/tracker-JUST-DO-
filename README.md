# tracker-JUST-DO-
Трекер привычек с социальным элеменом (JUST DO)
Учебный проект, разрабатываемый в рамках университетской практики. Приложение помогает пользователям формировать полезные привычки, отслеживать прогресс и поддерживать мотивацию.

## Стек технологий
- **Backend:** Python (FastAPI)
- **Frontend:** React js
- **База данных:** Supabase
