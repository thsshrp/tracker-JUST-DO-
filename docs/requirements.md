# 📋 Технические требования к проекту JUST DO

Мобильное веб-приложение для трекинга привычек с социальными элементами. Сочетает личный самоменеджмент с мотивацией через сообщество.

## 1. Роли пользователей
* **Обычный пользователь:** Повышение личной продуктивности (создание привычек, отслеживание выполнения, статистика).
* **Участник группы:** Мотивация через сообщество (взаимная поддержка, общая статистика).
* **Модератор группы:** Управление сообществом (администрирование, контроль контента).

## 2. Функциональные требования

### Аутентификация и профиль
* `FR1.1-1.3`: Регистрация по email/паролю, вход, восстановление пароля.
* `FR1.4-1.6`: Просмотр профиля, редактирование имени, загрузка аватара.

### Управление привычками
* `FR2.1-2.5`: Создание привычки (название, описание, категория, цель).
* `FR2.6-2.9`: Управление (просмотр списка, отметка выполнения, редактирование, удаление).
* `FR2.10-2.12`: Статистика (история, текущий стрик, общий процент выполнения).

### Функции групп и модерация
* `FR3.1-3.12`: Каталог групп, поиск, вступление (открытые/закрытые), лента прогресса, публикация постов (анонимно/открыто), лайки и комментарии.
* `FR4.1-4.11`: Создание группы (название, описание, тип), управление заявками, исключение участников, удаление контента.

## 3. Нефункциональные требования
* Интуитивный интерфейс с элементами геймификации.
* Защита конфиденциальности данных.
* Адаптивный дизайн для мобильных устройств и планшетов.
* Оптимизация времени загрузки.
